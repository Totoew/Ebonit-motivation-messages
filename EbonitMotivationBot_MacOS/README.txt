EBONIT MOTIVATION BOT - ИНСТРУКЦИЯ

КАК ЗАПУСТИТЬ ПРОГРАММУ:

1. Скачайте и распакуйте архив EbonitMotivationBot_MacOS.zip
2. Откройте папку EbonitMotivationBot_MacOS
3. Дважды кликните на файл "EbonitMotivationBot"
4. Если система предупредит о неизвестном разработчике:
   - Нажмите "Отмена"
   - Перейдите в Системные настройки → Защита и безопасность
   - Нажмите "Все равно открыть"

ТРЕБУЕМЫЕ ФАЙЛЫ:
- Excel файл с данными студентов в формате .xlsx
- VK API токен (получить: https://vk.com/dev/access_token)

ПРОБЛЕМЫ И РЕШЕНИЯ:
- Если приложение не запускается: откройте Терминал, перейдите в папку и выполните "./EbonitMotivationBot"
- Для первого запуска может потребоваться разрешение в Системных настройках

ВЕРСИЯ: 1.0
